Install dependencies and start the Expo web dev server.

Run the following steps in order:

1. Run `npm install --legacy-peer-deps` in the `python-book` directory and wait for it to finish.
2. Run `npx expo start --web` to launch the browser dev server.
3. Tell the user the app is running at http://localhost:8081 and remind them they can press `i` in the terminal to open the iOS Simulator instead.
